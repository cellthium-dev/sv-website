## Tech Stack

- **React 19** with TypeScript
- **TanStack Router** (file-based routing, auto-generated route tree)
- **Tailwind CSS 4** via Vite plugin
- **Vite** (using `rolldown-vite` as the build engine)
- **Biome** (linter + formatter) with the `ultracite` preset

## Architecture

This is a professional services landing page (SPA) for an independent expert in photovoltaic/energy systems.

### Routing

File-based routing via `@tanstack/router-plugin`. Routes live in `src/routes/`:

- `__root.tsx` — root layout
- `impressum/index.tsx` — legal page

The router auto-generates `src/routeTree.gen.ts` — do not edit this file manually.

### Components

All section components live in `src/components/` and are rendered in `src/App.tsx`. Each component is self-contained with its own hardcoded data (no external API calls). The exception is `BookingSystem.tsx`, which loads Calendly via an external script.

### Types

Shared TypeScript interfaces are in `src/types/index.ts`: `Badge`, `Service`, `Testimonial`, `Article`, `TimelineItem`, `FormData`, `BookingData`.

### Styling

Tailwind utility classes only — no CSS modules. Primary color: `#2563EB` (blue-600), secondary: `#004C99`. Responsive breakpoints: `sm`, `md`, `lg`.

### Code Standards (from Ultracite/Biome config)

- No `console.log` or `debugger` in committed code
- `async/await` over promise chains
- Early returns over nested conditionals
- No nested ternaries
- `rel="noopener"` on external links
- Avoid `dangerouslySetInnerHTML`
